# Coding Test Practice

## 소개
이 저장소는 **코딩 테스트 연습**을 위해 문제를 풀이한 내용을 정리한 공간입니다.  
현재 **백준(BOJ)** 과 **SWEA** 문제를 중심으로 학습하고 있으며,  
문제를 **난이도별로 분류**하여 체계적으로 풀이를 진행하고 있습니다.

모든 문제는 **Python**으로 풀이하고 있습니다.

---

## 학습 목표
- 다양한 유형의 알고리즘 문제 해결 능력 향상
- 난이도별 문제 풀이를 통한 단계적 실력 향상
- Python을 활용한 구현 능력 및 문제 해결력 강화
- 꾸준한 풀이 기록을 통한 코딩 테스트 대비

---

## 진행 방식
- **플랫폼**
  - 백준(BOJ)
  - SWEA

- **언어**
  - Python

- **분류 기준**
  - 난이도별 정리
  - 문제 유형별 학습
  - 플랫폼별 구분

---

## 풀이 원칙
- 가능한 한 **직접 풀이**를 우선으로 진행합니다.
- 문제를 해결한 뒤에는 더 나은 풀이 방법이 있는지 추가로 학습합니다.
- 같은 유형의 문제를 반복 학습하여 풀이 패턴을 익힙니다.
- 코드의 가독성과 효율성을 함께 고려합니다.

---

## 앞으로의 계획
- 난이도별 문제 풀이 개수 지속 확대
- 자주 출제되는 알고리즘 유형 집중 학습
- 풀이 코드와 함께 핵심 아이디어 정리
- 오답 문제 복습 및 재풀이 진행

---
