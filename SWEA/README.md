# SWEA Problem Solving

## 소개
이 디렉토리는 **SWEA 문제 풀이**를 정리한 공간입니다.  
SWEA 문제를 **난이도별로 분류**하여 학습하고 있으며,  
모든 문제는 **Python**으로 풀이하고 있습니다.

---

## 학습 방향
- SWEA의 난이도 체계를 기준으로 단계적으로 문제 풀이 진행
- 구현, 완전탐색, 그리디, DFS/BFS, DP 등 다양한 알고리즘 유형 학습
- 반복 풀이를 통해 문제 해결 속도와 정확도 향상

---

## 사용 언어
- Python

---

## 난이도 분류 예시
```bash
SWEA/
├── D1/
├── D2/
├── D3/
├── D4/
├── D5/
└── ...
```

---

## 풀이 기준
- 난이도별로 차근차근 문제를 풀이합니다.
- 문제를 푼 뒤에는 다른 풀이 방식도 함께 확인합니다.
- 자주 등장하는 유형은 따로 정리하여 복습합니다.
- Python 문법과 알고리즘 활용 능력을 함께 향상시키는 것을 목표로 합니다.
- 문법에 대한 공부 노트는 obsidian에 정리합니다. 

---

## 목표
- SWEA 주요 난이도 구간별 문제 풀이 경험 축적
- 코딩 테스트 실전 감각 향상
- 다양한 알고리즘 문제 해결 패턴 습득
- 안정적인 Python 풀이 능력 강화
